 Key Elements of a Sales Optimization Dashboard
1. Summary Cards (KPIs)
2. Total Sales

 Units Sold

Cities Covered

 Completion % (vs. targets)

Use bold fonts, icons, and conditional formatting for impact.

 2. Interactive Charts
Doughnut charts for completion percentages

Line/Column charts for monthly trends

Combo charts with secondary axis for Sales vs. Units

Make sure charts are linked to dynamic ranges or PivotTables.

3. Slicers & Filters
Filter by:

City

Product

Channel (Amazon, Flipkart, App)

Month

Ensure slicers are properly connected to all visuals.

4. Dynamic Formulas
Use OFFSET, COUNTA, and INDEX for dynamic ranges

Use IF, IFS, AND, OR for logic-based insights
